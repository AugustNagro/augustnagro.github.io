Master email with the following recommendations:

### General Strategy
Goal: Follow common email etiquette.

* Don't use gmail; switch to [Google Inbox](inbox.google.com)
* Start using Google Calendar. Primarily week view.
* Keep Inbox and Calendar as pinned tabs in Chrome.
* Install the Inbox phone app. Uninstall gmail, or turn off duplicate notifications.
* Add a profile photo.
* Learn and use Inbox's differentiating features.
* Remove any signature / footers ("sent from my iphone") in gmail settings.
* Use Reminders for todos that will be accomplished in a short amount of time. Use another service to keep track of long-term tasks.
* Aggressively clean your inbox by archiving (marking 'done' or swiping to right) all finished threads. If an inbox accumulates over 20 messages, the whole inbox needs to be [cleared](http://www.peterbronez.com/archive-all-messages-in-a-gmail-inbox).
* Use a secondary email account (Or search '10 minute mail') for signups on websites that may send spam. Unsubscribe from mailing lists and mark spam.
* Set default reply behavior to "Reply All" in Gmail settings. If people accidently reply to only you instead of the group, reply and then use the pop-out reply button (above trash can) to add the rest back into the thread. Let the transgressor know of his beginner mistake.

### Consolidating Accounts
Goal: Manage as much communication as possible from a single webpage.

* Have all external email forwarded to master account. "Forwarding and POP/IMAP" tab in gmail settings of non-master emails.
* Respond to a message with the same email address it was sent to. Enable "Send mail as:" in the master account settings ("Accounts and Import" tab). Select "Reply from the same address the message was sent to" option. 
* When composing a new message, select the relevent sender by clicking the "From xxxx@gmail.com" column.
* Setup Google voice or Hangouts to make phone calls.
* Use the right sidebar to send hangouts messages and make phone calls (click the "New Conversation" column, and then the phone icon) and ideally send SMS via [Project Fi](https://fi.google.com/about/).
* Take extra precautions to protect your account; use a long password and enable 2-factor authentication.

### Shortcuts
Goal: Speed up navigation by reducing number of clicks.

* Turn on keyboard shortcuts in gmail settings if disabled.
* View and learn shortcuts with Shift-?
* Learn Calendar’s shortcuts as well.
* Master advanced combinations like q-tab-enter-<contact's name>-down arrow-enter to send a text message to <contact's name>.
